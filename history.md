## history

```
# Zeigt die letzten 100 Befehle an
history
```

```
# !<Zahl>
# führt Befehl 24 aus der history nochmal aus
!24
```

```

```

!! 

# kombinierbar: macht ein sudo mit den letzten Befehl aus der history
